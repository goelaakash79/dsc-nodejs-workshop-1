# Steps to create a NodeJS app

> Setting up the project directory
 1. mkdir project_name
 2. cd project_name

 > Initializing your project
 3. npm init
	 Enter the project details that follows.
 * For Window Users:
	 4. echo.>app.js
 * For Linux Users:
	 4. touch app.js
5. Write server code in app.js file as given.

> Start server
6. node app.js
